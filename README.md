# Angular 15 Bank Learning

A mini banking UI to practice **Angular 15.2 (NgModule)** + **TypeScript** + **RxJS** with **Jest** unit tests.

Mini sistema bancário para praticar **Angular 15.2 (NgModule)** com **TypeScript**, **RxJS**, consumo de **APIs** e **Jest**.

> Objetivo: evoluir do zero ao especialista com entregas reais e testáveis.

## Features (incrementais)
- Visão de conta (saldo, dados do titular)
- Extrato / transações (listagem e detalhes)
- Transferência (Reactive Forms + validação de regras)
- Beneficiários (CRUD)
- Rotas (guards, resolver, lazy loading)
- Camada de API (HttpClient + interceptors)
- Testes unitários com Jest (services/components/forms)

## Stack
- Angular **15.2**
- TypeScript
- RxJS
- Jest (unit tests)

## Como rodar
```bash
npm install
ng serve
